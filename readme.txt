# Hotel Booking Management System

## Overview
This is a Flask-based Hotel Booking Management System, designed to streamline the process of managing hotel bookings, reservations, and customer information. The application provides a user-friendly interface for customers.

## Features
- User Authentication: Secure user authentication system for customers.
- Booking Management: Efficient booking system with the ability to view bookings.
- Room Availability: Real-time room availability status to prevent double bookings.
- Customer Information: Manage customer details and history of their bookings.

## Technologies Used
- Flask: Web framework for Python.
- MySQL: Database for storing application data.
- flask_mysqldb: Flask extension to simplify MySQL database connections.
- HTML/CSS: Front-end styling and structure.
- Bootstrap: Front-end framework for responsive design.
